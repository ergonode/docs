# Language


| Key           | Type |Description  |
|---------------|------|--------------|
| id       | uuid      |  Language id  |
| code     |  string   |   Language code  |
| name     |  string   |  Language name |
| active    |  bool  |        Is language active |


**Example**

```
{
        "id": "a1083663-61ad-4c1c-b979-a5aec82574df",
        "code": "DE",
        "name": "German",
        "active": false
    },
```
