# Element properties


| Key           | Type |Description  | Required |
|---------------|------|--------------|----------|
| attribute_id    |  uuid | Attribute Id         | Yes      |
| required    | bool   | Is attribute required       | Yes      |

**Example**

```
    {
        "attribute_id": "e3f236d1-0e52-5804-be5b-096bbe225fda",
        "required": true
    }
```
